# **App Name**: Bloodflow

## Core Features:

- Dashboard Overview: Display summary cards for blood stock, donor count, and pending requests.
- Donor Management: Add and remove donor records with local storage.
- Request Management: Submit blood requests and track their status (pending/fulfilled).
- Inventory Tracking: Manage blood stock by blood group with editable entries.
- Analytics Dashboard: Display interactive charts (Pie + Bar) for data visualization.
- Settings Panel: Toggle dark/light mode, reset data, and preview themes.

## Style Guidelines:

- Primary color: Light violet (#A78BFA) for a modern and vibrant feel.
- Background color: Desaturated blue-gray (#232F41) for a dark, futuristic look.
- Accent color: Sky blue (#38BDF8) to highlight key interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look.
- Floating animated icons from React-Icons for navigation, using a gentle floating animation.
- Responsive grid layout with glassmorphism cards, soft shadows, and glowing borders for a semi-3D effect.
- Framer Motion for page transitions, card hover effects (scale up to 1.05, shadow deepens, background blur intensifies).
